## 🛠️ **Development Ready**

### **For New Developers**
```bash
# Clone the repository
git clone https://github.com/sanudasandipa/Uploder-mobileapp.git
cd Uploder-mobileapp

# Run automated setup
chmod +x setup.sh
./setup.sh
```

